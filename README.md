# Temperature Monitoring Package

Denne repository samler åbne tekniske ressourcer og eksempler til monitorering af temperatur i køle-, fryse- og medicinskabe i kommunal drift.

Repositoryet er under opbygning og vil løbende blive udvidet med:

- Grafana dashboards
- Payload decoders
- Sensor-eksempler
- Implementeringsguides
- Visualiseringer
- Dokumentation

## Formål

Formålet er at understøtte deling og genbrug mellem kommuner gennem åbne og praksisnære IoT-komponenter.

## Målgruppe

Kommuner, driftspersonale, IoT-medarbejdere og andre offentlige organisationer med interesse for temperaturmonitorering og datadrevet drift.
